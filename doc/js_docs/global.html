createQM_Document()

This function will create a new Realtime document in the user's Google Drive with the name input into a text box. This function is only executed if the user goes to the web app without a valid document id in the url.
Preconditions:
The user did not navigate to an existing valid Realtime document.
Postconditions:
A new Realtime file is created in the users Google Drive with the name input into the text box.
